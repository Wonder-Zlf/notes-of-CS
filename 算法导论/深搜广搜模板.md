# 深搜广搜模板

1、深搜递归

```
public void dfs01() {
    判断{
        计算；
        dfs01();
    }
}
```


2、深搜辅助栈

```
public void dfs02() {
    while(栈非空){
        取出栈顶元素；
        判断{
            计算；
            入栈；
        }
    }
}
```


3、广搜队列

```
public void bfs() {
    while(队列非空){
        取出队首元素；
        判断{
            计算；
            入队；
        }
    }
}
```
